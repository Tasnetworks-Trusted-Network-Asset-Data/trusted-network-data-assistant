# Trusted Network Data Assistant

## Problem
Network planners need reliable cable asset data to determine underground cable thermal ratings. Data is currently scattered across design documentation, asset records and rating tables.

## Proposed Solution
A prototype that extracts cable properties from design documents, compares them against asset records and rating tables, flags discrepancies, and presents a single trusted view for planner review.

## Demo Flow
1. Upload or select a design sheet
2. Extract cable properties
3. Compare with GIS/SAP asset record
4. Identify missing or conflicting values
5. Show trusted planner view with evidence and confidence
